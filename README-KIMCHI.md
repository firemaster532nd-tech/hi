# 🐕 김치의 세상 - 말티즈 홈페이지

귀여운 말티즈 김치를 위한 개인 애완동물 홈페이지입니다.

## 📋 프로젝트 개요

김치는 2020년 3월 15일에 태어난 사랑스러운 말티즈입니다. 이 웹사이트는 김치의 일상, 추억, 성장 과정을 기록하고 공유하기 위해 만들어졌습니다.

## 🌟 주요 기능

### 1. 홈페이지 (kimchi-home.html)
- 김치 소개 히어로 섹션
- 프로필 카드 (기본 정보)
- 최근 갤러리 미리보기
- 특별한 순간 하이라이트

### 2. 김치의 역사 (kimchi-history.html)
- 타임라인 형식의 성장 기록
- 태어난 날부터 현재까지의 이야기
- 주요 마일스톤 및 특별한 순간들
- 성장 통계 및 정보

### 3. 갤러리 (kimchi-gallery.html)
- 카테고리별 사진 필터링 (일상, 여행, 친구들, 특별한 날)
- 메이슨리 레이아웃의 사진 갤러리
- 영상 모음
- 사진/영상 업로드 기능
  - 드래그 앤 드롭 지원
  - 파일 미리보기
  - 메타데이터 입력 (제목, 날짜, 카테고리, 설명)

### 4. 일상 (kimchi-daily.html)
- **일기장**: 날짜별 일상 기록
- **일정 관리**: 
  - 다가오는 일정 (예방접종, 미용, 생일 등)
  - 미니 캘린더
- **건강 기록**:
  - 체중, 검진 기록
  - 예방접종 일정
  - 상세 건강 기록 타임라인

### 5. 소개 (kimchi-about.html)
- 김치 상세 소개
- 성격 및 특징
- 좋아하는 것 / 싫어하는 것
- 하루 일과
- 말티즈 품종 정보
- 연락처 및 SNS

## 🚀 사용 방법

1. 웹 브라우저에서 `kimchi-home.html` 파일을 엽니다
2. 상단 네비게이션 메뉴를 통해 각 페이지로 이동할 수 있습니다
3. 갤러리 페이지에서 사진/영상을 업로드할 수 있습니다
4. 일상 페이지에서 일기, 일정, 건강기록을 관리할 수 있습니다

## 📸 사진 추가하기

현재는 사진 자리 표시자(placeholder)가 표시됩니다. 실제 사진을 추가하려면:

1. `images/` 폴더에 사진을 저장합니다
2. HTML 파일에서 해당하는 `.image-placeholder` 또는 `.gallery-placeholder` 부분을 수정합니다
3. 예시:
```html
<!-- 기존 -->
<div class="image-placeholder">
    <i class="fas fa-dog fa-10x"></i>
    <p>김치의 사진을 여기에 올려주세요</p>
</div>

<!-- 변경 후 -->
<img src="images/kimchi-photo.jpg" alt="김치" style="width: 100%; border-radius: 20px;">
```
