## 📐 **object-position 설명**

### 기본 개념
```
object-position: [수평] [수직];
```

### 수직 위치 값
```
0%   ← 이미지 최상단
↓
30%  ← 상단에서 30% (슬라이더용)
↓
35%  ← 상단에서 35% (갤러리용)
↓
50%  ← 정중앙 (기본값)
↓
100% ← 이미지 최하단
```

---

## 🐶 **왜 30~35%인가?**

### 강아지 사진의 특징
```
┌─────────────────┐
│   [얼굴/머리]   │ ← 30~35% 위치
│                 │    (가장 중요한 부분)
│   [몸통]        │
│                 │
│   [다리/꼬리]   │
└─────────────────┘
```

### 효과
- ✅ **얼굴이 중앙**에 위치
- ✅ 표정이 선명하게 보임
- ✅ 다리 부분이 잘려도 자연스러움
- ✅ 전체적으로 균형잡힌 구도

## 💡 **작동 원리**

### object-fit: cover
- 이미지 비율 유지
- 영역을 완전히 채움
- 넘치는 부분 자동 크롭

### object-position: center 35%
- 수평: 중앙 (center)
- 수직: 상단에서 35% 지점
- 해당 지점이 박스 중앙에 배치

### 결과
```
┌─────────────┐        ┌─────────────┐
│             │        │   [얼굴]    │ ← 35% 지점
│  원본 이미지│   →   │   [몸통]    │ ← 박스 중앙
│             │        │   [다리]    │
│             │        └─────────────┘
└─────────────┘        (크롭된 영역)
```

## 🔧 **수동 조정 방법**

이미지마다 강아지 위치가 다를 경우:

### 방법 1: 전역 값 변경
```css
/* /home/user/webapp/css/kimchi-style.css */

.gallery-item img {
    object-position: center 40%; /* 더 아래쪽 */
}
```

### 방법 2: 개별 이미지 조정 (HTML)
```html
<img 
    src="image.jpg" 
    style="object-position: center 25%;"
>
```

### 추천 값 범위
- **20-30%**: 얼굴 특화 (클로즈업)
- **30-40%**: 얼굴+몸통 (전체샷)
- **40-50%**: 몸통 중심 (액션샷)
