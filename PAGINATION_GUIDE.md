# 📄 갤러리 페이지네이션 기능

## ✅ 구현 완료

사진첩 섹션에 페이지네이션 기능이 추가되었습니다!

---

## 🎯 주요 기능

### 1. 카드 개수 제한
- **페이지당 9개 카드**로 고정
- 최신 업로드 사진부터 표시
- 9개 초과 시 자동으로 다음 페이지 생성

### 2. 페이지네이션 컨트롤
```
[ ← 이전 ]    1 / 3 (총 25장)    [ 다음 → ]
```

- **이전/다음 버튼**: 페이지 이동
- **페이지 번호**: 현재 페이지 / 전체 페이지
- **사진 개수**: 필터링된 총 사진 개수

### 3. 필터와 연동
- 필터 버튼 클릭 → 해당 카테고리의 첫 페이지로 이동
- 자동으로 사진첩 섹션으로 스크롤
- 각 카테고리별로 독립적인 페이지네이션

## 💻 작동 방식

### 1. 필터 버튼 클릭
```javascript
// 전체보기 클릭
필터: all → 모든 사진 표시 (최신순 9개)
페이지: 1

// 일상 클릭
필터: daily → '일상' 카테고리만 표시 (최신순 9개)
페이지: 1 (리셋)

// 미용사진 클릭
필터: grooming → '미용사진' 카테고리만 표시
페이지: 1 (리셋)
```

### 2. 페이지 이동
```javascript
// 다음 버튼 클릭
현재 페이지: 1 → 2
표시 사진: 10~18번 사진

// 이전 버튼 클릭
현재 페이지: 2 → 1
표시 사진: 1~9번 사진
```

### 3. 자동 스크롤
```javascript
// 필터 변경 또는 페이지 이동 시
window.scrollTo({ top: 0, behavior: 'smooth' });
```

## 📊 데이터 흐름

### 렌더링 프로세스
```
1. galleryManager.renderPhotos(filter, page)
   ↓
2. 필터링: 카테고리에 맞는 사진만 선택
   ↓
3. 정렬: 최신순으로 역순 정렬
   ↓
4. 슬라이싱: (page-1) * 9 ~ page * 9
   ↓
5. DOM 생성: 9개 카드 생성
   ↓
6. 페이지네이션 UI 업데이트
```

### 예시 (총 25장, 페이지 2)
```
전체 사진: [1, 2, 3, ..., 25]
      ↓
필터링 (all): [1, 2, 3, ..., 25]
      ↓
역순 정렬: [25, 24, 23, ..., 1]
      ↓
슬라이싱 (페이지 2): [16, 15, 14, 13, 12, 11, 10, 9, 8]
      ↓
표시: 9개 카드
```

## ✅ 테스트 시나리오

### 1. 사진 업로드 테스트
1. 10장 이상 업로드
2. 전체보기에서 9개만 표시 확인
3. 페이지네이션 UI 표시 확인
4. "다음" 버튼으로 2페이지 이동
5. 나머지 사진 확인

### 2. 필터 테스트
1. "일상" 필터 클릭
2. 일상 카테고리만 표시 확인
3. 9개 이상이면 페이지네이션 표시
4. 다른 필터로 전환 → 페이지 1로 리셋

### 3. 페이지네이션 테스트
1. "다음" 버튼 클릭 → 2페이지 이동
2. 10~18번 사진 표시 확인
3. "이전" 버튼 클릭 → 1페이지 이동
4. 1~9번 사진 표시 확인
5. 첫 페이지에서 "이전" 버튼 비활성 확인
6. 마지막 페이지에서 "다음" 버튼 비활성 확인
